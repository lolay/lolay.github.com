We specialize in Location-Based Social Networking applications for mobile devices such as:  iOS, Android, Windows Phone 7, Blackberry and Mobile Web.

Welcome to our github page. We have some great projects to share with you:

{grid} https://github.com/lolay/grid

A Grid View that was loosely based on the DTGridView but allows insets per row and column. In addition, it can be resized without reloading all data. This works similar to TableViewCell in that it supports the ability to enqueue and dequeue cells to preserve memory.

{http-client} https://github.com/lolay/http-client

Based on 0.5 http client from google code http://code.google.com/p/objective-c-http-client/ with a lot of customization

{strands} https://github.com/lolay/strands

Library for handling threading and queues within iOS. Both with blocks and without (i.e. iOS 3.1)

{investigo} https://github.com/lolay/investigo

Tracker library for AdMob, others to come soon (Flurry, etc.). - Async Admob reporting using NSURLConnection delegate methods - Once-per-app Admob reporting using Docs dir

{imaji} https://github.com/lolay/imaji

Image library for manipulating images and image views in objective-c

{pair} https://github.com/lolay/pair

A library for handling key value pairs.

{kat} https://github.com/lolay/kat

Lolay library that adds functionality typically found in UI Kit (i.e. UI Kit Kat :)

{erred} https://github.com/lolay/erred

Anything to be done with error handling on the iOS

{bigben} https://github.com/lolay/bigben

Objective-C libraries for dealing with date/time.

{locksmith} https://github.com/lolay/locksmith

iOS Wrapper for Keychain Utilities such as SecItemCopyMatching, SecItemAdd, SecItemUpdate and SecItemDelete.

{cocoa-amf} https://github.com/lolay/cocoa-amf

CocoaAMF is a set of classes which can make AMF0 and AMF3 remoting calls or act as a server to handle AMF requests. Based on: - Deusty Designs for CocoaAsyncSocket (http://code.google.com/p/cocoaasyncsocket/) - Binary God for BGHUDAppKit (http://code.google.com/p/bghudappkit/) - Jonathan Wight for TouchJSON (http://code.google.com/p/touchcode/) - The PyAMF Team for PyAMF (http://pyamf.org/) and therefor letting me have a reference and unit tests which I can steal shamelessly

{identitas} https://github.com/lolay/identitas

Objective-C library for handling identity
